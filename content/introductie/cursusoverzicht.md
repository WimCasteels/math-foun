# Cursusoverzicht

## Doel van de cursus

Deze cursus behandelt de wiskundige fundamenten voor machine learning, verdeeld over vier hoofdthema's:

| Thema | Onderwerpen |
|-------|-------------|
| **Lineaire Algebra** | Vectoren, matrices, PCA |
| **Analyse** | Afgeleiden, optimalisatie, gradient descent |
| **Waarschijnlijkheid & Statistiek** | Kansen, verdelingen, MLE/MAP, onzekerheid |

## Waarom wiskunde voor ML?

Machine learning algoritmes zijn gebaseerd op wiskundige concepten:

| ML Concept | Wiskundige Basis |
|------------|------------------|
| Data representatie | Vectoren & matrices |
| Training | Optimalisatie (gradient descent) |
| Voorspellingen | Kansverdelingen |
| Model evaluatie | Statistiek & intervalschatting |
